## 🌟 Características

- 🖼️ **Interfaz moderna** con tema oscuro profesional
- 📝 **Validación de datos** inteligente (campos obligatorios, numéricos)
- 💾 **Exportación a Excel** automática con timestamp
- 👀 **Vista previa** de datos ingresados
- 🎨 **Diseño responsive** con grids y frames optimizados
- 🔧 **Zero config** - solo Python y librerías estándar

## 🚀 Instalación Rápida

```bash
# Instalar dependencias
pip install pandas openpyxl

# Ejecutar la aplicación
python excel_data_app.py
````
